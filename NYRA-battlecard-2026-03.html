What They Sell

Cowork is a feature inside the Claude Desktop app that gives Claude access to a folder on your computer. It can read, edit, create, and organize files. Users set a task, Claude plans and executes, optionally in parallel sub-agents. It produces professional deliverables (DOCX, XLSX, PPTX). Expandable via MCP connectors and a growing plugin marketplace. Available on Pro ($20/mo), Max ($100-200/mo), Team ($30/user/mo), and Enterprise plans.

NYRA Desktop — Key Differentiators

NYRA Desktop is a self-hosted, multi-provider AI operating system for power users and developers who want full control over their AI workflow.

1. Multi-Provider Freedom

The only desktop AI tool that lets you use Claude, GPT, Ollama, and custom providers in the same app. Switch models per-task. No vendor lock-in. A/B test models side by side with the built-in comparison tool.

2. 40+ Built-in Tool Panels

Git integration, integrated terminal, Task Board (Kanban), API Playground, Diff Viewer, Build Validator, Agent Studio, Agent Analytics, Knowledge Graph, Performance Profiler, Report Generator, Webhook Manager, and 25+ more — all native, no plugins needed.

3. Offline-First Architecture

Everything runs locally on Electron + SQLite. Offline queue syncs when connectivity returns. Use Ollama for fully air-gapped operation. No cloud dependency, no data leaves your machine.

4. 5-Tier Memory System

Working → Episodic → Semantic → Procedural → Archival memory persists across sessions. Knowledge Graph visualizes relationships. Unlike every competitor, NYRA actually remembers your project context.

5. Full Cowork + Agent Orchestration

Cowork mode with progress tracking, folder management, plan views, and agent sections. Agent Pipeline for multi-step orchestration. Agent Studio for building custom agents. Agent Analytics for performance tracking.

6. Cross-Platform + Free

macOS, Windows, Linux. Self-hosted with BYO API keys — no subscription fee, no rate limits imposed by the platform, no waitlist. You control the cost.

7. Consistent Warm Theme System

Cohesive 5-color design palette (terra, gold, sage, blush, warm) with 12 wallpaper textures, dark/dim/light modes, and custom theme editor. Professional, distinctive aesthetic that competitors' generic UIs can't match.
